PocketFare - Mumbai

Features

  • Supports Fare Calculation for Taxis and Auto-Rickshaws in Mumbai.
  • Night-fare automatically enabled during 12am to 5am.
  • Fare calculated on the fly as you type.
  • Check for Fare revision updates and use the latest taxi and rickshaw fares as soon as they are revised by the Mumbai Metropolitan Region Transport Authority (MMRTA).
  • Works on every phone that supports Java (CLDC 1.1 and MIDP 2.0)

Screenshots

Minimum Phone Configuration

  • Any Phone that supports Java with CLDC 1.1 and MIDP 2.0
  • GPRS/3G/Wi-Fi connection required for fares update.

Is there an Android/iPhone version of Pocket-Fare Mumbai?

Although there are no plans to create an iPhone port, the PocketFare Android App is expected to be completed by Fall 2011.

Is there a PocketFare app for other Indian cities?

Not yet, but I want to release PocketFare apps for as many Indian cities as possible. I can't seem to find any official resources that contain the fare charts for other cities, so if you come across these documents for your city, send them to me and I'll create a PocketFare app for your city.
